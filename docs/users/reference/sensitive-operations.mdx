# Sensitive operations

The following mutations are sensitive operations and typically require [consent](/users/concepts/consent). Some may not, depending on context (for example, `suspendAccountMembership`).
All links lead to the mutation's API Reference page.
Mutations marked with the symbol ⮂ are eligible for [server-to-server consent](/users/concepts/consent/server-to-server).

## Accounts and onboarding {#sensitive-accounts}

- [`closeAccount`](https://api-reference.swan.io/mutations/close-account/)
- [`openAccount`](https://api-reference.swan.io/mutations/open-account/) ⮂
- [`finalizeAccountHolderOnboarding`](https://api-reference.swan.io/mutations/finalize-account-holder-onboarding/) ⮂
- [`finalizeOnboarding`](https://api-reference.swan.io/mutations/finalize-onboarding/) (deprecated) ⮂
- [`updateAccountSettings`](https://api-reference.swan.io/mutations/update-account-settings/) ⮂
