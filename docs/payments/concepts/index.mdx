# Payments concepts

<p className="ia-lede">The models behind Swan payments: how a payment groups transactions, the payment methods that move money, and the merchant side that accepts it.</p>

## How payments work

A payment is a request to move money. Each payment groups one or more transactions, which are what actually credit and debit accounts.
