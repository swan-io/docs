# Transactions

An account is essentially a list of transactions, and transactions directly impact account balances.

## Transaction statuses {#transaction-statuses}

There are six possible statuses for Swan transactions.
Each payment method uses a **different combination** of these statuses with specific flows.
Refer to the schemas for [credit transfers](/topics/payments/credit-transfers/#statuses), [direct debits](/topics/payments/direct-debit/#statuses), and [cards](/topics/payments/cards/#statuses) for more information about statuses for that payment method.
