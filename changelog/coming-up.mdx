<details>
  <summary>⚠️ **Upcoming breaking changes**</summary>

  <div>

    📆 On **April 30, 2026**, the deprecated `InternalCreditTransfer` value will be removed from the [`PaymentProduct`](https://api-reference.swan.io/enums/payment-product/) enum. Remove any references to this value from your transaction queries and filtering before this date.

    📆 On **May 20, 2026**, the [Consent Notification endpoint](/topics/users/consent/guide-configure-notifications) timeout will be reduced from 3 seconds to 2 seconds. If your integration uses in-app consent notifications, make sure your endpoint responds within 2 seconds. Otherwise, Swan falls back to text message. OAuth2 authentication is not affected.

    📆 On **May 21, 2026**, public onboarding URLs will switch to [Onboarding v2](/preview/new-onboarding/) for all new onboardings, and the v1 public onboarding interface will be retired. Action is only required if you prefill onboarding fields via the API before redirecting users.

    📆 On **September 30, 2026**, [Onboarding API v1](/preview/new-onboarding/legacy-onboarding/) will be shut down and all v1 endpoints will no longer be available. Migrate to [Onboarding v2](/preview/new-onboarding/) before this date.
