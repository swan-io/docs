# Closing accounts

<p className="ia-lede">Understand who can close a Swan account, how each closure process works, the reason codes Swan assigns, and how end users are notified by email.</p>

A Swan account might need to be closed for several reasons.
**You**, your **account holders** and **account legal representatives**, **Swan**, and **third parties** can all request an account be closed.

You only pay for accounts with the status `Opened`.
As soon as an account enters the closure process, the status changes to `Closing` and you're no longer charged for the account.

:::info Account statuses
Refer to the [account statuses section](/accounts/concepts/account/statuses) for an overview of what happens when an account is `Closing` or `Closed`.
:::
