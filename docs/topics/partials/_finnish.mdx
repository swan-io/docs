Finnish is a supported [**account language**](../accounts/index.mdx#language) and [**account membership language**](../accounts/memberships/index.mdx#language) with certain limitations:

<ul>
    <li>Finnish isn't available as a [**card language**](../cards/index.mdx#language). When the account language is Finnish, the card language defaults to English, which includes card packaging and the language displayed on payment terminals.</li>
    <li>Finnish isn't available for the [**bank details document**](../accounts/index.mdx#bank-details). When the account language is Finnish, the bank details document is generated in English.</li>
</ul>
