# Get information about multiple users

You can **call the API** to get information about multiple users within your project

Identification information is also available on your **Dashboard** > **Data** > **Users**.

:::tip Prerequisites
You need a **project access token** call the query.
:::

## Guide {#guide}

1. Call the `users` query.
1. Add all information you'd like to review. The sample query retrieves user IDs, when the users joined the project, their identification status, and the language they used to authenticate, as well as the total count of users in the project.
1. Add [pagination](../../../developers/using-api/pagination.mdx) to get only the most recent PVID result (line 9-11).

```graphql {5-7,9,13} showLineNumbers
query InfoMultipleUsers {
  users {
    edges {
      node {
        id
        joinedAt
        identificationStatus
        authenticators {
          acceptLanguage
        }
      }
    }
    totalCount
  }
}
```
