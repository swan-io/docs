# Get card package tier

Card products are associated with a specific [card package](/topics/cards/card-packages/): **Standard**, **Essential** and **Premium**. You can retrieve the package tier for a specific card using the `card` query.

## Retrieve card package tier {#guide-main}

1. Call the `card` query.
1. Enter the `cardId` (line 2).
1. Add `cardPackage` to receive the card package tier (line 4).

<h3 id="query-main">Query</h3>

<a href="https://explorer.swan.io/?query=cXVlcnkgUmV0cmlldmVDYXJkUGFja2FnZSB7CiAgY2FyZChjYXJkSWQ6ICIkWU9VUl9DQVJEX0lEIikgewogICAgaWQKICAgIGNhcmRQYWNrYWdlCiAgfQp9Cg%3D%3D&tab=api">Open in API Explorer</a>

```graphql {2,4} showLineNumbers
query RetrieveCardPackage {
  card(cardId: "$YOUR_CARD_ID") {
    id
    cardPackage
  }
}
```
<h3 id="payload-main">Payload</h3>

The response includes the `cardPackage` (line 5), which is the `Premium` tier.

```json {5} showLineNumbers
{
  "data": {
    "card": {
      "id": "$YOUR_CARD_ID",
      "cardPackage": Premium
    }
  }
}
```
