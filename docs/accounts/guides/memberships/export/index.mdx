## What's included in an export

Every export contains the same fields, regardless of path:

<ul className="ia-shared-list">
  <li>Membership ID, email, role, and permissions</li>
  <li>Account ID and account name</li>
  <li>Invitation status (pending, accepted, suspended, or disabled)</li>
  <li>Created and last-updated timestamps</li>
  <li>Binding errors, if any</li>
</ul>

## Before you start

<ul className="ia-shared-list">
  <li>You need admin or read access to the account.</li>
  <li>Exports are scoped to one account at a time.</li>
</ul>
