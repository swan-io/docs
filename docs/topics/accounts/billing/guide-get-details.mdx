# Get invoice details

Get details about a single invoice with the API.

You can also review each invoice on your **Dashboard** > **Data** > **Accounts**.
Open the account for which you want to review invoices, then go to the **Invoices** tab and find the invoice you'd like to review.

## Guide {#guide}

1. Call the `accountInvoice` query.
1. Add the invoice ID.
1. Add all information you'd like to review about the invoice (lines 3-16).
