<p className="ia-lede">When an account's <Term id="legal-representative">legal representative</Term> requests closure, initiate the process: send them a Dashboard link, or start it with the API.</p>

:::caution Suspended accounts
Only Swan can act on `Suspended` accounts. If the [account status](/accounts/concepts/account/statuses) is `Suspended`, you can't initiate closure.
:::
