## What's included in an export

Every export contains the same fields, regardless of path:

<ul className="ia-shared-list">
  <li>Membership ID, email, permissions, and legal-representative status</li>
  <li>Account ID and account name</li>
  <li>Invitation status (pending, accepted, suspended, or disabled)</li>
  <li>Created and last-updated timestamps</li>
  <li>Binding errors, if any</li>
</ul>

## Before you start

<ul className="ia-shared-list">
  <li>You need admin or read access to the account.</li>
  <li>Exports are scoped to one account at a time.</li>
</ul>

## After you export

After you finalize your export, whether from your Dashboard or using the API, a link to download the export is **sent to you by email**.
You must be **logged in** to download the `.csv` export file.

Download links are **valid for seven days**, after which you need to generate a new export.

:::warning Sharing the download link
If you forward someone the email with the download link, they can only use it if they also have access to the Dashboard.
Otherwise, an error occurs.

Instead, consider downloading the document and sending it to them.
:::
