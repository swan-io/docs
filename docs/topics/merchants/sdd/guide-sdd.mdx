# Accept payments with SEPA Direct Debit

## Step 1: Add a merchant profile {#step-1}

Before accepting payments with SEPA Direct Debit, you need to add a merchant profile.

→ [Add a merchant profile](../profiles/guide-add.mdx)

→ [Update a merchant profile](../profiles/guide-update.mdx) *if needed*

## Step 2: Request the payment method {#step-2}

→ [Request the SEPA Direct Debit payment method](./guide-request.mdx)

## Step 3: Declare payment mandates {#step-3}

Declare your payment mandates to Swan, mandatory for both Core and B2B schemes.

→ [Declare a payment mandate](./guide-declare.mdx)

## Step 4: Initiate transactions {#step-4}

Call the API to initiate SEPA Direct Debit transactions so merchant can accept payments from their clients.
